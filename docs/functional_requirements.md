# アプリケーション機能要件・UI/UX設計仕様書

## 1. 概要
本仕様書は、小学生中学年（3〜4年生）を対象とした学習Webアプリケーションの機能要件、学習テーマ、UI/UX方針、および学習データの保存・分析基盤仕様を定義するものである。

---

## 2. ターゲットユーザーおよび設計方針

* **ターゲットユーザー**: 小学生中学年（3〜4年生 / 8〜10歳）1名（※シングルユーザー前提。ログイン認証は不要）
* **開発における基本姿勢**:
  * **実装速度最優先**: リッチすぎる演出や重いグラフィックを避け、コアな学習機能と分析の高速構築を優先。
  * **軽量サクサク動作**: Raspberry Pi 4のスペック制限でも遅延なく快適に動き、ストレスを与えないUI/UX。
* **アプリケーションの目的**:
  * 算数・英語・タイピングの基礎力強化と学習習慣の定着。
  * データベースに学習履歴（正答率・解答速度・プレイ時間等）を蓄積し、客観的な学習分析を提供。
  * 自身の成長や得意・苦手傾向を視覚化することで、本人の学習意欲（モチベーション）を向上させる。

---

## 3. 主要学習テーマとコンテンツ要件

### 3.1 算数（Math）
* **対象・学習テーマ**:
  1. **繰り上がり足し算**: 桁の繰り上がりを含む2桁以上4桁以下の足し算（例: `348 + 574`）
  2. **繰り下がり引き算**: 桁の繰り下がりを含む2桁以上4桁以下の引き算（例: `1024 - 458`）
  3. **拡大部分九九・掛け算**: 1×1 から 12×12 までの掛け算（例: `12 × 11 = 132`）
  4. **指定解のマッチング問題**: 提示された答え（例: 12）になる掛け算を選択・回答する問題（例: 「答えが12になる式をえらぼう → 3×4, 2×6」）
  5. **逆算・穴埋め問題**: 答えから式を考える問題（例: `x × 3 = 6` のときの `x` を求める）
* **ゲーム・学習形式**:
  * タイムチャレンジ形式（一定時間内に何問解けるか）
  * ステップアップレベル（レベル別問題発生）

### 3.2 英語（English）
* **コンテンツ内容**:
  * 基礎英単語（アルファベット、身近な単語・イラストマッチング）
  * リスニング＆発音チェック（音声の聞き取りと正しいスペル選択）
  * 簡単な日常フレーズ・会話文

### 3.3 タイピング（Typing）
* **コンテンツ内容**:
  * 英単語タイピング（英語学習と連携したローマ字・英単語入力）
  * 英文タイピング（短い英文の入力・スピード測定）
* **機能**:
  * リアルタイムWPM（Word Per Minute）計測、正確性（Accuracy %）算出、入力ミスの視覚的フィードバック。

## 4. 学習結果の保存・分析基盤仕様

### 4.1 データ保存項目（SQLiteデータベース構造要件）
学習結果の分析およびモチベーション向上のため、以下のデータをセッションごとに保持する。

1. **学習セッションデータ**:
   * ユーザーID、学習日時、学習テーマ（算数/英語/タイピング）、プレイ時間
2. **パフォーマンス指標**:
   * 正解数・不正解数・正答率（%）
   * 平均解答速度（秒/問）、タイピングWPM、正確性（%）
   * 誤答ログ（どの問題で間違えたか）
3. **成長・達成度データ**:
   * 累積ポイント・連続学習日数（ストリーク）
   * 獲得バッジ/トロフィー一覧

### 4.2 モチベーション向上・分析機能（Analytics & Motivation UI）
* **成長グラフ・マイページ**:
  * 過去1週間・1ヶ月の「スコア推移」「学習時間」「タイピング速度向上グラフ」を表示。
* **得意・苦手分析**:
  * 「算数のわり算が得意！」「英語の単語タイピングの精度がアップ！」など、ポジティブな分析フィードバック。
* **ゲーミフィケーション要素**:
  * 学習を続けるとランクアップするレベルシステム。
  * 苦手な問題を復習できる「リベンジモード（弱点克服クイズ）」。

## 5. UI/UX設計ガイドライン (React + Tailwind CSS)

1. **ビジュアルデザイン**:
   * 小学生中学年がワクワクするダッシュボード風デザイン（ネイビー/アクアブルー/アクセントイエローなどの洗練されたモダンUI）。
   * レスポンシブ対応（PCキーボード操作・タブレット操作対応）。
2. **タイピング＆キーボードUI**:
   * 画面上にオンスクリーンキーボードまたは入力ガイドを表示し、正しい指の位置や入力をサポート。
3. **即時フィードバック**:
   * 正解・好記録達成時のモーダル＆紙吹雪エフェクト（Canvas Confetti）。
   * 得点アップ時の効果的なアニメーション（Framer Motion / Tailwind Animation）。

---

## 6. 画面一覧および画面遷移

```
[ トップ / マイページ ] ---+---> [ 算数ゲーム ] ---------> [ 学習結果・分析 ]
                        |---> [ 英語ゲーム ] ---------> [ データベース保存 ]
                        |---> [ タイピングゲーム ] ---> [ グラフ・成長可視化 ]
                        +---> [ 保護者・分析レポート ]
```

1. **トップ / マイページ (`/`)**: 学習テーマ選択、本日の目標、現在のレベル、ストリーク（連続日数）。
2. **算数・英語・タイピング学習画面 (`/study/:subject`)**: 各テーマのゲーム・問題プレイ画面。
3. **リザルト画面 (`/result/:sessionId`)**: 今回のスコア、WPM/正確性、獲得ポイントの表示。
4. **学習分析ダッシュボード (`/analytics`)**: 得意・苦手傾向、成績推移グラフ、達成バッジ一覧。
